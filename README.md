# KrishiMitra AI — Premium 3D Agriculture Dashboard

**KrishiMitra AI** is a state-of-the-art agricultural management and intelligence co-pilot designed for modern farming. Built using a nature-inspired aesthetic (emerald greens, sleek glassmorphism, responsive grids, and smooth animations), the platform empowers farmers and agronomists with 3D telemetry visualization, real-time crop disease detection, weather intelligence, and an interactive AI advisory system.

### 🌐 Live Production Deployment
🚀 **Live URL:** [https://krushimitra-ai-two.vercel.app](https://krushimitra-ai-two.vercel.app)

---

## ✨ Features

### 1. 🔑 Premium Authentication Flow
- **Glassmorphic Login**: Secure credentials input panel with interactive signup/signin transitions.
- **One-Click Sandbox Login**: Jump straight into the dashboard with a pre-configured demo sandbox profile.
- **Visual Accent**: Backed by a grid-textured ambient layout with floating leaf particles.

### 2. 📊 3D Plot Telemetry Console
- **3D Interactive Model**: Rendered using React Three Fiber with restricted orbiting cameras (prevents disorienting loops).
- **Active Hotspots**: Clickable plot sensors highlighting soil moisture, crop species, and heat emitters.

### 3. 🔬 Crop Disease Scanner
- **Live Video Streaming**: Streams device camera feeds directly into a glassmorphic scanner card.
- **AI Scan Simulator**: Neural laser scan animation assessing crop blight.
- **Advisory Reports**: View confidence metrics, severity scores, organic treatment lists, and export report files (.txt logs).

### 4. ☀️ Weather Intelligence
- **Animated SVG Weather Elements**: Handcrafted custom animations reflecting real-time climatic conditions.
- **Live Gauges**: Tracks relative humidity, wind speed, barometric pressure, and UV index.
- **7-Day Outlook**: Responsive forecast list tied to a daily details selector card.

### 5. ⚙️ Farm Operations Center
- **Emitters Controller**: Live-toggles zone water drips with status feedback.
- **NPK Nutrient Levels**: Soil Nitrogen, Phosphorus, and Potassium gauges.
- **Yield Charts**: Visual actual-vs-target yield outputs.

### 6. 💬 Conversational Co-pilot (Floating Bot Widget)
- **Breathing AI Avatar**: Pinned globally at the bottom-right corner with notification badges.
- **Quick Chat & Full Advisor Mode**: Processes queries regarding blights, companion crops, or soil pH directly.

## 🛠️ Technology Stack
- **Framework**: Next.js 15 (React 19 App Router)
- **Styling**: Tailwind CSS v4 + PostCSS
- **Database / Auth / Storage**: Supabase (supabase-js SDK)
- **AI Engine**: Gemini Pro / Vision APIs (google-generative-ai SDK)
- **Animations**: Framer Motion
- **3D Engine**: Three.js / React Three Fiber / Drei
- **Icons**: Lucide Icons
- **Deployment**: Vercel

## 🚀 Local Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Safal-48/PS-3-_gdgnagpur_-Team-Perplexity-.git
   cd PS-3-_gdgnagpur_-Team-Perplexity-
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run in development mode**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
